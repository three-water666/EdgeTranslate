-   Improved extension UI layering on pages that use dialogs, popovers, or other top-layer UI, so the selection button, screenshot selection, and translation result panel can stay visible when needed while normal pages keep the lighter rendering path. PR #21
-   Migrated the PDF viewer to versioned PDF.js assets and syncs runtime files from `pdfjs-dist`, making PDF translation builds and future viewer upgrades easier to maintain. PR #23, PR #26
-   Isolated the Google Translate Element runtime into versioned snapshots and refreshed it to TE_20260513, with build output patched to use extension-local runtime paths. PR #24
