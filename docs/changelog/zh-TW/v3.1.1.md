### 新增

-   新增「滾動一頁後自動關閉結果面板」版面配置選項。啟用後，翻譯結果面板會在頁面滾動約一個視窗高度後自動關閉，減少繼續閱讀時的遮擋。

### 修復

-   修復長按翻譯觸發後 `mouseup` 事件被過度攔截的問題，降低對頁面自身點擊、選取和互動邏輯的影響。
-   修復設定頁版面配置區域的開關對齊問題，在新增開關後保持更穩定的排版。

### 維護

-   發布流程改為使用人工整理的三語言 changelog，確保 GitHub Release 內容與本地發布說明一致。
-   建置時的 lint 範圍調整為只檢查 JavaScript 測試檔案，避免非目標測試資源影響打包流程。
