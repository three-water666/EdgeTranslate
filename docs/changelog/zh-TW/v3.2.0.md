### 新增

-   改進 OCR 設定頁的語言包狀態展示，將「已下載」和「已啟用」拆成獨立標記，並增加手動上傳對話框，作為內建下載失敗時的備用安裝方式。
-   在彈出視窗和設定頁提示本機 PDF 翻譯需要開啟瀏覽器的檔案 URL 存取權限，並在 README 中補充相關設定步驟。PR #22

### 改進

-   改進頁面彈窗、對話框和 popover 場景中的擴充功能 UI 層級，使劃詞按鈕、截圖選取區和翻譯結果面板能在需要時顯示在頁面頂層，同時普通頁面保持輕量渲染。PR #21
-   將 PDF 檢視器遷移到版本化的 PDF.js 資源，並從 `pdfjs-dist` 同步執行時資產，提升 PDF 翻譯建置和後續升級的可維護性。PR #23, PR #26
-   將 Google Translate Element 執行時隔離到版本化快照，並刷新到 TE_20260513，建置時會改寫為擴充功能本機資源路徑。PR #24

### 修復

-   修復透過劃詞按鈕觸發發音時沒有重用目前選中文字的問題。PR #21
-   修復固定位置翻譯面板在頁面捲動時仍可能自動關閉的問題。PR #28

### 維護

-   從依賴套件生成 OCR 執行時檔案，移除舊的靜態 OCR 執行時副本，並補充相關維護文件。PR #25
-   升級 axios 和建置相關依賴，新增傳遞依賴覆寫，修復依賴審計中的安全問題。PR #27
-   用 Node 腳本替換舊的 gulp 擴充功能建置流程，改進開發模式下的資源監聽和重新建置邏輯，並清理 pnpm 鎖定檔。PR #28
