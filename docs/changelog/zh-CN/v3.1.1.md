### 新增

-   新增「滚动一页后自动关闭结果面板」布局选项。启用后，翻译结果面板会在页面滚动约一屏距离后自动关闭，减少阅读后续内容时的遮挡。

### 修复

-   修复长按翻译触发后 `mouseup` 事件被过度拦截的问题，降低对页面自身点击、选择和交互逻辑的影响。
-   修复设置页布局设置区域的开关对齐问题，在新增开关后保持更稳定的排版。

### 维护

-   发布流程改为使用人工整理的三语言 changelog，确保 GitHub Release 内容与本地发布说明一致。
-   构建时的 lint 范围调整为只检查 JavaScript 测试文件，避免非目标测试资源影响打包流程。
