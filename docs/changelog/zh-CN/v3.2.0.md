### 新增

-   改进 OCR 设置页的语言包状态展示，将“已下载”和“已启用”拆成独立标记，并增加手动上传对话框，作为内置下载失败时的备用安装方式。
-   在弹窗和设置页提示本地 PDF 翻译需要开启浏览器的文件 URL 访问权限，并在 README 中补充相关设置步骤。PR #22

### 改进

-   改进页面弹窗、对话框和 popover 场景中的扩展 UI 层级，使划词按钮、截图选区和翻译结果面板能在需要时显示在页面顶层，同时普通页面保持轻量渲染。PR #21
-   将 PDF 查看器迁移到版本化的 PDF.js 资源，并从 `pdfjs-dist` 同步运行时资产，提升 PDF 翻译构建和后续升级的可维护性。PR #23, PR #26
-   将 Google Translate Element 运行时隔离到版本化快照，并刷新到 TE_20260513，构建时会改写为扩展本地资源路径。PR #24

### 修复

-   修复通过划词按钮触发发音时没有复用当前选中文本的问题。PR #21
-   修复固定位置翻译面板在页面滚动时仍可能自动关闭的问题。PR #28

### 维护

-   从依赖包生成 OCR 运行时文件，移除旧的静态 OCR 运行时副本，并补充相关维护文档。PR #25
-   升级 axios 和构建相关依赖，添加传递依赖覆盖，修复依赖审计中的安全问题。PR #27
-   用 Node 脚本替换旧的 gulp 扩展构建流程，改进开发模式下的资源监听和重新构建逻辑，并清理 pnpm 锁文件。PR #28
